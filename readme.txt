
- Funkcionalnosti:
	1. CRUD Repository,
	2. CRUD Issue,
	3. CRUD User,
	4. Comment Issue,
	5. Close-Open Issue

- Opis
	User moze da kreira svoj Repository. 
Jedan repository moze da ima vise user-a koji ucestvuju na Issue. 
Takodje user moze da kreira svoj issue u okviru repository koji mu je dodeljen.
Issue moze da bude otvoren i zatvoren., User moze da komentarise issue.
Postoji registracija usera.
